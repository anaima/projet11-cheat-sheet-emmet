1. HTML

Il existe sur Atom (éditeur de texte) une manière de taper encore plus rapidement les balises pour écrire un code.
Pour le fichier HTML, on a (cf les images ci-dessus):

  • ul>li*5 :(ul) (li)(/li) (li)(/li) (li)(/li) (li)(/li) (li)(/li) (/ul)
  • div+p+bq :(div)(/div) (p)(/p) (blockquote)(/blockquote)
  • nav>ul>li :(nav) (ul) (li)(/li) (/ul) (/nav)
  • #header :(div id="header")(/div)
  • .class :(div class="class")(/div)
  • meta:vp :(meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0" /)
    • Pour voir encore plus de raccourcis
